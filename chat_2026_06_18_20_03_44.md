```markdown

This document lists the automated agents, bots, or scripts used in this project, along with their purposes, configurations, and interactions.

---
```

### **Key Questions to Address in `AGENTS.md`**
1. **What agents exist?**  
   - List each bot, script, or automated workflow (e.g., GitHub Actions bots, Slack bots, CI/CD agents).

2. **What do they do?**  
   - Clearly state their purpose (e.g., "This bot enforces code style checks").

3. **How are they configured?**  
   - Link to config files or environment variables (e.g., `BOT_API_KEY=xxx`).

4. **How do they interact?**  
   - Document dependencies between agents (e.g., "Agent X must run before Agent Y").

5. **How to debug?**  
   - Log locations, permissions required, etc.
